# Day 6 - AWS Cloud Intro

## What is the Cloud?
The cloud is someone else's computers that you rent over the internet. You pay only for what you use instead of buying expensive servers.

## The 3 AWS Services I Learned Today

### S3 - Simple Storage Service
Like Google Drive but for developers. Used to store files, datasets, images, and AI model outputs.

### EC2 - Elastic Compute Cloud
A virtual computer in the cloud. You can rent a powerful computer by the hour, run your AI model on it, then turn it off when done.

### Lambda - Serverless Functions
Run code without managing a server. It only runs when triggered and you only pay for those seconds.

## Why AI Engineers Use AWS
- Store huge datasets for AI training
- Run AI models on powerful cloud computers
- Serve AI responses to millions of users 24/7
- Scale automatically based on demand
